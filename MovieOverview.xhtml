This application is created by : Elwaleed Elmana (H00241623) , Mohamed Bast( H00200406) ,Weam Hameed( H00245135)

Our application will give you what you want to know about movies,where we create filters by year, directors, actors/actress, ratings. Plus we create top movies, recommendations user preferences page.

Our web app consist of three pages:

Home page, this page created by Elwalled K Elmana, The home page contain an overview of all the new movies filtered by two main filters "Daily Tracker" and " popular" with there subcategories. used bootstrap CSS framwork including themes and JS style, with feature like: 1-Toggle collapsed navigation bar header 2- Drop down navigation menu 3- buttons and forms with bootstrap themes. 4- Main page comes with columns that represent the Movies with link to MovieSite

The movie page, is created by Mohamed Baset , that shows an overview of the movie, pic, movie trailer, cast information, lead actor, year, genre...etc.External files used: two image files and an embedded YouTube video.As well as the developer used bootstrap CSS framework plus custom style to override standard bootstrap style. In addation, he used jQuery to hide and show some div elements when clicking on different navigation button.

In the actors page ,which is created by Weam Hameed, information about thousands of actors and actress can be found . you can search about any actors you want. It gets it's information from the movie data base back service, using JSON format. the page contains update automatically by using AJAX. the client page will run throw web browser and its written by HTML, CSS, and java script.
